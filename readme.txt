基于WeUI和Vue(2.x)开发的移动端UI组件库，主要服务于微信页面

初始化项目：

npm install vue-cli -g // 如果还没安装
vue init airyland/vux2 projectPath

cd projectPath
npm install --registry=https://registry.npm.taobao.org
npm run dev
